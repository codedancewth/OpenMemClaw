## 字段说明

### 必填字段
- `生成信息` - 所有子字段
- `基础信息.称呼` - 至少有一个称呼
- `核心项目` - 至少一个项目（如果有）

### 可选字段
- `基础信息` 其他字段
- `行为特征` - 可根据数据丰富度选择
- `偏好与兴趣` - 根据提取结果
- `人际关系` - 如有相关信息
- `记忆规则` - 如用户有明确表达
- `统计摘要` - 增强信息
- `原始数据引用` - 用于追溯

## 置信度标注

对于 AI 推测的信息，使用以下格式：

```markdown
### 推测信息（置信度：70%）
- 编辑器：VS Code（从代码讨论推断）
```

置信度级别：
- **高 (80-100%)**：用户明确陈述
- **中 (50-79%)**：多次出现或强烈暗示
- **低 (<50%)**：单次提及或弱暗示，需确认
